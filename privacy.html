Privacy Policy for Email Writer

Effective Date: October 2025

Email Writer is a Chrome extension that helps users generate AI-powered email replies directly within Gmail. The extension is designed to respect user privacy and collect the minimum data necessary for core functionality.

Data Collection and Usage

  • When a user requests an AI-generated email reply, the text entered or selected (such as the email body and tone preference) is securely transmitted to our backend API for processing.
  • The backend temporarily uses this input to generate a response using a language model, then returns it immediately to the extension. No personal data, message content, or generated responses are stored or shared.
  • The extension may use Chrome’s local storage only to save recent generation history for the user’s convenience. This data remains entirely on the user’s device and is not transmitted anywhere.

Permissions

The extension uses minimal permissions:

  • Storage — to save user preferences (e.g., tone and generation history) locally on the device.
  • Host permissions — to communicate securely with Gmail and the AI backend API.

The extension does not access or modify your Gmail messages beyond what is necessary to generate replies, and does not collect browsing history or personal data.

Data Security

All communication between the extension and the backend occurs over secure HTTPS. Data sent for processing is ephemeral — it is used only for the duration of the request and then discarded.

User Control

You may remove the extension at any time through your browser’s settings. Once removed, it no longer has access to Gmail or any related data.
